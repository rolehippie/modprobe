## Requirements

- Minimum Ansible version: `2.10`

## Default Variables

### modprobe_extra

List of extra modules to load

#### Default value

```YAML
modprobe_extra: []
```

### modprobe_general

List of general modules to load

#### Default value

```YAML
modprobe_general: []
```
